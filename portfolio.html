    Ride Share Rails

    This project builds a Rails app that manages the Rideshare program with the following requirements:

    Navigation functionality: As a user, I want a consistent, easily accessible navigation menu with different buttons/links from any page.

    Passenger functionality On the passenger list page, I want to click on a passenger to see that passenger's details.

    Driver functionality: On the driver list page, I want to click on a driver so that I can see that driver's details.

    Video Rental Store

    This final full-stack project will be something that a rental store employee might use to manage the video store where they work.

    The React front-end implementation will interact with a Rails API on the back-end. The Rails API wraps an external API that contains many endpoints related to videos. The external API is The Movie DB.

    The end product allows users to search for a video, select a video, add a video to the video library, and see a list of customers. The end product should be deployable to Heroku.

    Links: Github Repo & Live Demo

    Exquisite React App

    This project is a single app that will allow users to contribute to a poem one line at a time using a form. Each line of poetry is a piece of data that propagates from component to component.

    A user filling out the form may only see the most recent line of poetry submitted. When the users are finished submitting lines of poetry, they should be able to see the entire poem.
